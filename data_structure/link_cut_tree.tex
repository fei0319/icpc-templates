\subsection{动态树}

\subsubsection{用法}

\begin{itemize}
\item \lstinline{Mono}：一个幺半群或半群类。需重载 \lstinline{*} 运算符。
\item \lstinline{N}：一个整数，值应\textbf{大于}需要使用的最大下标。
\end{itemize}

该实现包含的 Splay 可单独抽出作为平衡树使用。

\subsubsection{注意}

下标从 0 开始，但不支持使用 0 下标。

\subsubsection{性能}

$n, q = 2\times 10^5$ 约花费 1.28 个标准时间。
